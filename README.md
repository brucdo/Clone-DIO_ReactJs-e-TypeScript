## Available Scripts
In the project directory, you can run:

### To run "back-end":
`npm run api`

  Home: 
  http://localhost:8001

  Resources:
  http://localhost:8001/users

### To run "front-end":
`npm start`

  Home: 
  http://localhost:3000

  Routes: 
  * /feed    
  * /login
  * /register
